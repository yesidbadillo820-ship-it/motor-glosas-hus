💰 Tarifas contratadas por EPS

Base de conocimiento para validar glosas por tarifas contra valores pactados en contratos.

📊 Subir Excel de tarifas RECOMENDADO

Sube directo el Excel que envía la EPS (tipo Famisanar) con 3 hojas: Anexo 3 (Servicios CUPS con SOAT ± %), Anexo 3.1 (Medicamentos valor fijo) y Anexo 3.2 (Suministros con IVA). El sistema auto-detecta EPS, contrato y vigencia desde el encabezado.

📤 Opción alternativa: subir CSV manual

Columnas: eps, cups, descripcion, valor, contrato, modalidad, vigencia_desde, vigencia_hasta. Separador tab, coma o punto-y-coma. Todas las filas se guardan como VALOR_FIJO.
